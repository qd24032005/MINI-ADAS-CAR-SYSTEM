# H-TH-NG-PH-T-HI-N-V-C-NH-B-O-TI-N-VA-CH-M-CHO-XE-T-L-I---MINI-ADAS
Mini ADAS sử dụng STM32F401RE để mô phỏng xe tự hành phát hiện và cảnh báo tiền va chạm. Hệ thống đo khoảng cách bằng HC-SR04, hiển thị trạng thái SAFE/WARN/DANGER trên LCD và LED, điều khiển motor qua TB6612, hỗ trợ Auto Mode và Manual Mode qua Bluetooth HC-06.
